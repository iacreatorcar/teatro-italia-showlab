Avvio progetto in locale

Requisiti

  • Node.js installato (versione compatibile con Next.js del progetto)

Prima volta (o dopo modifiche a package.json)

npm install

Avvio server di sviluppo

npm run dev

Apri il browser su http://localhost:3000

Altri comandi utili

  • npm run build — build produzione
  • npm run start — avvia build produzione (va fatto npm run build prima)
  • npm run lint — controllo lint del codice
