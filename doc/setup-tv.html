Setup fisico TV (4 schermi)

Perché non uno switch HDMI

Uno switch HDMI mostra una sola sorgente per volta, si commuta manualmente tra le TV. Con 4 TV che devono mostrare contenuti diversi/simultanei serve un output indipendente per ogni schermo.

Soluzione: un dongle/mini PC per TV

Ogni TV ha il proprio dispositivo collegato in HDMI, connesso alla rete (wifi o ethernet), che apre il browser in modalità kiosk puntato su:

https://tuosito.netlify.app/tv?screen=1
https://tuosito.netlify.app/tv?screen=2
https://tuosito.netlify.app/tv?screen=3
https://tuosito.netlify.app/tv?screen=4

Il numero screen=N corrisponde allo screen_id assegnato nel pannello admin (/admin).

Setup Raspberry Pi (Chromium kiosk mode)

  1. Installa Raspberry Pi OS (versione Lite o Desktop)
  2. Installa Chromium: sudo apt install chromium-browser
  3. Crea script autostart kiosk in ~/.config/autostart/kiosk.desktop: 
    [Desktop Entry]
Type=Application
Name=Kiosk
Exec=chromium-browser --noerrdialogs --kiosk --incognito https://tuosito.netlify.app/tv?screen=1
  4. Disabilita screensaver/sospensione: xset s off; xset -dpms; xset s noblank
  5. Riavvia, verifica apertura automatica in fullscreen

Setup Fire TV Stick / Chromecast Google TV

  1. Installa app Fully Kiosk Browser (Amazon Appstore / Play Store)
  2. Configura URL di avvio: https://tuosito.netlify.app/tv?screen=1
  3. Attiva modalità kiosk (fullscreen, no barra, autostart al boot)
  4. Disattiva screensaver/sospensione nelle impostazioni dispositivo

Decisione finale

Da valutare insieme al cliente in base a: budget, necessità di affidabilità h24, competenze tecniche disponibili per manutenzione.
